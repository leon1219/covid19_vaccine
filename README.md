# Covid-19_vaccine Twitter data

These data files are used to support the findings in the paper "Dynamic assessment of the COVID-19 vaccine acceptance leveraging social media data" published in the Journal of Biomedical Informatics by Lingyao Li, Jiayan Zhou, Zihui Ma, Michelle Bensi, Molly Hall, and Gregory Baecher.

Restrictions apply to the availability of Twitter data, which are used under license for the current study, and so specific user information or tweets messages are not publicly available. The tweet IDs are however available from this published dataset, but the original tweets are available with permis-sion of the Twitter Inc.
